# is-cpf
Determine if a value is higher than a value

## Install
```
$ npm install --save is-min
```

## Usage
```js
const isMin = require('is-min');

console.log(isMin('Hello World!', 6));
//=> true

console.log(isMin('Hello', 10));
//=> false
```
